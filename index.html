Robox

A low-cost, computer-based Robotics Platform opening the access to technology education.

About the Robox

The Robox is a low-cost computer-based Robotics Platform

It combines an easy robotics programming interface with a full fledged computer -- the Raspberry Pi.

Robox can be programmed via Python with the Turtle Bot Api and Scratch for maximal curricular compatibility.

Features requests and development will be hosted on Github.
